  БЫСТРЫЙ СТАРТ (VPS сервер)
══════════════════════════════════════════════════════════════════

1. Установите Node.js 20:
   curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
   sudo apt install -y nodejs

2. Распакуйте архив:
   mkdir -p /var/www/chess
   cd /var/www/chess
   tar -xzf chess-game.tar.gz

3. Установите PM2:
   sudo npm install -g pm2

4. Запустите:
   cd /var/www/chess/standalone
   pm2 start ecosystem.config.js
   pm2 save
   pm2 startup

5. Для домена настройте Nginx (см. nginx.conf)
